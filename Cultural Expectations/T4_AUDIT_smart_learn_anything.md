## BUT — the drift the user sensed is real and locatable

**The original T4 is a claim about a BELIEF APPLIED TO PEOPLE:** *"smart people are expected to learn
everything."* The question is *why we hold that belief about individuals in software but not in
medicine/law/accounting.*

**What §4 (the best-evidenced leg) actually answers is a DIFFERENT question:** *why did some fields
install external accountability brakes and software didn't?* That is a claim about **fields and
institutions**, not about the belief-applied-to-people.

So T4 quietly migrated **belief-about-people → structure-about-fields.** The structure leg is true and
well-evidenced, but it explains the *consequence* (software lacks external brakes), not the *belief
itself* (why "just learn it" is culturally credible for software). The belief leg survives only:
- partially in **§5b** (title-as-compression-artifact — the closest thing to a belief mechanism), and
- as **framing** in §0 / B§0,
- and is deliberately **un-sentenced** per the TAXONOMY fence (correctly — the belief is described/traced,
  never judged).

**Net: the belief is DESCRIBED (taxonomy), TRACED (§5b title mechanism), and its CONSEQUENCE explained
(§4 structure) — but the question "why is the belief PLAUSIBLE for software specifically?" has no
first-class answer.** That is the gap the user pointed at. It is not that T4 vanished; it is that the
*belief-plausibility* leg was never built, and the structure leg grew to fill the space.

## What this session's work offers the missing leg (candidate, NOT yet added)

The three-axis / craft-decay findings (C7, verified; 2.1c) supply a **belief-plausibility mechanism**
that did not exist when T4 was written:

> **"Smart people can learn anything" is grantable to SOFTWARE because software's craft CHURNS and
> CANNOT BE BANKED (C7, Majors: ~2–3yr decay onset, ~5yr ceiling).** When the substrate resets every
> few years, *everyone is perpetually re-learning* — the twenty-year senior engineer is ALSO re-learning
> — so "a smart person can learn it" looks **true**, because seniority confers no un-crossable moat.
> In MEDICINE/LAW/ACCOUNTING the craft **compounds and is banked** (C7: finance craft is bankable) —
> the expert's twenty years ARE an un-crossable moat — so "just learn it" is visibly **false**.

This is the SAME craft property (churn / non-bankability) that makes the CTO unable to bank its mode
(axis 3 / mode-separability). One substrate-churn fact would then explain BOTH:
- why the CTO can't bank its craft (role distinctiveness, 2.1c), and
- why "smart → learn anything" is culturally credible for software (T4 belief-plausibility).

**Status: CANDIDATE.** Attractive because it (a) re-centers T4 on the belief, (b) unifies T4 with the
session's verified CTO work, (c) is falsifiable. Cautions before adopting:
- **Fence check (TAXONOMY §4):** this mechanism EXPLAINS why the belief is held — it must stay a
  *description/trace* of the belief, NOT a verdict on it. "The belief is plausible because craft churns"
  is description (safe); "therefore the belief is correct/incorrect" would sentence the Belief row
  (forbidden). Keep it descriptive.
- **It's a belief-PLAUSIBILITY claim, not a belief-TRUTH claim.** Churn makes "just learn it" *look*
  true; whether it IS true (can a smart person actually learn it) is a separate, unmade claim — do not
  conflate.
- **Evidence base:** C7 craft-decay is verified for CTO (Majors, named author) and the CFO-banks
  contrast is sourced; extending "compounds/banks" to medicine & law specifically is inference (the
  surgery lit from Run C touches it but wasn't framed as craft-bankability). Flag as 2-of-N.
- **Does not rescue §4/§5 from their gaps** (G-2 master-variable demotion, G-5 Fayol-lineage gap stand).

## Recommendation (for user decision — not yet executed)

Add a T4 belief-plausibility sub-claim (the churn/bankability mechanism) as a NEW leg, explicitly:
- placed as **description/trace** under the taxonomy fence (never a verdict on the belief),
- scoped as belief-*plausibility* not belief-*truth*,
- flagged 2-of-N on the compounds-side evidence,
- cross-linked to C7 / 2.1c / axis-3 as the unifying substrate-churn fact.

This re-lands the original thesis ON THE BELIEF (where the user wanted it) while keeping the
well-evidenced structure leg (§4) as the *consequence* explanation. The two then read as: **§4 = why
the brake is absent (structure); new leg = why the belief is credible (craft-churn); §5 = the title
that carries it.** Together they cover belief → plausibility → consequence without a master variable.
